## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

You will need Expo to run this project

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:louisegn/airbnb-clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
